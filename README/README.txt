Web application (django). An excellent phone directory for the organization. It is presented as an adaptive web application with the ability to search by name, email or phone number. When you enter the directory, you are presented with the structure of the organization and you can select a specific department and it will expand and present you with contacts. Or type in the search what you are looking for. Administration of the application is implemented through the administrative panel. The ability to collapse and expand table rows is implemented using javascript/css and the binding goes to the names of departments registered in the template home.html .
